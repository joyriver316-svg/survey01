# A.aura 온톨로지 직결용 응답코드 ID (Entity ID) 매핑 명세서

본 문서는 안티그래비티 설문 완료 후, 유저의 점수와 선택지를 A.aura 지식 그래프 DB(SPARQL 쿼리 엔진)가 즉시 인식할 수 있도록 시스템 Entity ID로 변환하여 출력 세션에 바인딩하기 위한 마스터 백엔드 규칙입니다.

## 1. 바우만 4대 진단축 점수 구간별 ID 매핑 (4단계 스케일)
안티그래비티 계산기(Formula)에서 최종 합산된 점수 구간에 따라 유저 세션에 해당 Entity ID를 매핑합니다.

### [O/D 축] 유수분 밸런스 (접두어: SKIN.OD)
- **점수 조건:** `O_score` - `D_score` 연산값 기준
  - `+35점 이상`: `SKIN.OD.O_HIGH` (지성 강함)
  - `+1점 ~ +34점`: `SKIN.OD.O_MID` (지성 약함)
  - `-1점 ~ -34점`: `SKIN.OD.D_MID` (건성 약함)
  - `-35점 이하`: `SKIN.OD.D_HIGH` (건성 강함)
- **특이 로직:** 두 점수가 모두 35점 미만이면서 자가진단(Q10)이 복합성(3번)일 경우 `SKIN.OD.O_MID` 부여 및 `COMBO` 태그 동시 활성화.

### [S/R 축] 민감도 (접두어: SKIN.SR)
- **점수 조건:** `S_score` 총점 기준 (Q9 가산점 포함)
  - `45점 이상`: `SKIN.SR.S_SEVERE` (민감성 매우 강함)
  - `30점 ~ 44점`: `SKIN.SR.S_MILD` (민감성 약함)
  - `15점 ~ 29점`: `SKIN.SR.R_MILD` (저항성 약함)
  - `14점 이하`: `SKIN.SR.R_STRONG` (저항성 강함)

### [P/N 축] 색소성 (접두어: SKIN.PN)
- **점수 조건:** `P_score` 총점 기준
  - `40점 이상`: `SKIN.PN.P_HIGH` (색소성 강함)
  - `25점 ~ 39점`: `SKIN.PN.P_MILD` (색소성 약함)
  - `24점 이하`: `SKIN.PN.N_LOW` (비색소성)

### [W/T 축] 주름/탄력 (접두어: SKIN.WT)
- **점수 조건:** `W_score` 총점 기준
  - `40점 이상`: `SKIN.WT.W_HIGH` (주름/탄력 저하 강함)
  - `25점 ~ 39점`: `SKIN.WT.W_MILD` (주름/탄력 저하 약함)
  - `24점 이하`: `SKIN.WT.T_HIGH` (탄력 팽팽함/지속성 강함)

## 2. 행동, 성향, 안전 필터 문항 1:1 매핑 규칙
유저가 특정 선택지를 마킹하거나 입력했을 때 즉시 최종 패킷에 포함시켜야 하는 고유 고정 ID 목록입니다.

| 문항 No | 유저 응답 조건 | 시스템 반영 고유 Entity ID | 온톨로지 매핑 노드 | 알고리즘 반영 목적 |
| :--- | :--- | :--- | :--- | :--- |
| **Q9** | ①~⑤ (아토피, 지루성 등) 중 선택 | `FILTER.SAFE_MODE.ATOPIC` | SafetyFilter | **최우선 안전모드 락:** 추천 성분 하드 가이드 적용 |
| **Q38**| 특정 회기 성분 선택 시 | `ING.ALLERGY.{성분명}` | Ingredient_Allergy | **Hard Filter:** 해당 성분 전성분 포함 제품 무조건 배제 |
| **Q39**| 특정 알레르기 성분 선택 시 | `ING.ALLERGY.{성분명}` | Ingredient_Allergy | **Hard Filter:** 해당 성분 전성분 포함 제품 무조건 배제 |
| **Q40**| ①, ②, ③ (향 알레르기/자극) 선택 | `ING.ALLERGY.FRAGRANCE` | Ingredient_Allergy | **Hard Filter:** 인공향료/에센셜오일 포함 제품 원천 배제 |
| **Q42**| 복수 선택된 고기능성 성분명 | `ING.USED.{성분명}` | Ingredient_Used | **Soft Weighting:** 유저 경험 성분 기반 학술 리터러시 튜닝 |
| **Q22/23**| 임신, 수유, 생리 주기 변동 선택 | `MOD.PREGNANT` / `MOD.MENSES` | ContextModifier | **Modifier:** 결과 화면 주의 안내 문구 캐비엇 노출 |
| **Q27**| 스트레스 지수 Likert 5점 이상 | `MOD.STRESS_HIGH` | ContextModifier | **Modifier:** 스트레스성 트러블 케어 루틴 가중치 부여 |
| **Q34**| 선호하는 제형 선택 시 | `FORM.SERUM` / `FORM.CREAM` 등 | ProductForm | **Soft Weighting:** 추천 포트폴리오 상단 노출 우선순위 |
| **Q46**| 선호하는 향 카테고리 선택 시 | `FRAGRANCE.{향유형}` / `FRAGRANCE.NONE`| Fragrance Type | **Soft Weighting:** 선호 향기 제품 가점 매칭 |
| **Q44**| 월평균 화장품 지출 금액 선택 | `SPEND.T1` ~ `SPEND.T6` | Spend Tier | **Price Filter:** 유저 지출 수준 타깃팅 제품 프라이싱 필터 |
| **Q45**| 화장품 지속 사용 기간 선택 | `REPURCHASE.EXPLORER` / `LOYALIST`| Repurchase Behavior | 구매 행동 성향 세그먼트 매핑 |
| **Q43**| 성분 확인/검색 여부 선택 | `LIT.L1` / `LIT.L2` / `LIT.L3` | Literacy Level | 결과 레포트 텍스트 깊이(Depth) 및 문장 난이도 조절 |
| **Q47**| 진단을 통해 가장 기대되는 점 | `EXPECT.EXPERT_RECO` 등 | Expectation Value | 결과 화면 최상단 개인화 웰컴 마케팅 카피 바인딩 |

## 3. 데이터 신뢰도 및 시스템 품질 지표 (Consistency Index)
- **조건:** Q1-Q11 응답 일치도 및 Q4-Q10 교차 검증 점수가 **70% 미만**일 때
- **시스템 반영 ID:** `CONFIDENCE.LOW`
- **라우팅 규칙:** 결과 화면 로드 시 하단에 **"일관되지 않은 응답 패턴이 감지되었습니다. 정확한 분석을 위해 1:1 전문가 상담 혹은 재진단을 권장합니다."** 팝업 레이어 강제 트리거.
